Responsive website


In this project i worked with learning how to make a responsive website. For planning it I learned how to use different preparation tools such as moodboard, styletile and wireframe. These changed a little bit over the course of making the whole project, as my ideas kept evolving. I got assigned the style "retro videogame design", which was quite a bit of a challenge.

A responsive designs means that the design should be able to fit and scale it self to all different screensizes fron desktop to mobile. For making this website responsive, I have chosen to use mediaqueries.

It was very important for me, to prepare the base of the design of my website. All the tools I learned to use before coding the website was some really big guidelines for me to follow, and have become essential for me when building a webpage, cause it gives some great guidelines to follow.

After using all the preparation tools it was time to code everything in HTML and CSS. As I tend to use alot of time on my computer already, I had a really easy time understanding the coding language that is HTML & CSS.
