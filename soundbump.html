Soundbump - Json project


For this project we had to work with creating a website about a common passion in our group. We chose to go with the topic music.

The challenge of this project, was working with Json and how to fetch data from a file made in RESTDB.
When fetching this data, we have applied loopview and singleview in Javascript.

We also made our own logo in Illustrator, and edited our frontpage in Photoshop.
Everything else is made in HTML and styled with CSS.

!DISCLAIMER!
The database we used has had some problems since this project was finished, and therefore some pictures might not appear in loop and- singleview.
